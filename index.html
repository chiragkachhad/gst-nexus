How to use (any software)

  1. Books: Export purchase register for the month from Tally / Zoho / Busy / Marg / SAP / Excel. Need GSTIN, Invoice No, Date, Taxable, IGST/CGST/SGST (headers auto-mapped).
  2. GSTR-2B: gst.gov.in → Returns Dashboard → GSTR-2B → Download Excel or JSON.
  3. GSTR-2A (optional): Same portal → GSTR-2A → Download for three-way nexus / supplier follow-up.
  4. Choose mode → Run Reconciliation → export Audit Excel for CA file.
  5. ONLY_IN_BOOKS = invoice only in your books → cannot claim yet (ask supplier to file). IN_2A_NOT_2B = in 2A but not yet in 2B → wait. MATCHED_2B = can claim (other Sec 16 conditions apply).
